- Текстовые документы с критериями и ссылкой на тест.

## Как работать с файлами .ipynb

У вас есть два основных способа работы с блокнотами Jupyter:

### Вариант 1: Google Colaboratory (рекомендуется)
1. Откройте файл `.ipynb` по ссылке с Google Диска.
2. Нажмите кнопку **«Открыть в Colab»**.



3. **Сохраните копию** на свой Google Диск (Файл → Сохранить копию на Диске) или скачайте файл.
4. В своей копии вы можете:
   - Создавать новые ячейки: через меню **Вставка** или кнопку **+ Код**.
   - Запускать код:
     - Кнопка запуска в ячейке.
     - Сочетание клавиш `Ctrl + Enter`.
     - Меню **Выполнить → Выполнить все**.

---

### Вариант 2: Локальная работа в VSCode

#### Установка ПО (Windows)

1. **Скачайте и установите VSCode:**
   - Ссылка: [https://code.visualstudio.com/download](https://code.visualstudio.com/download)
   - **Важно:** при установке отметьте галочки:
     - *Зарегистрировать Code в качестве редактора для поддерживаемых типов файлов*
     - *Добавить в PATH*

2. **Скачайте и установите Python (версия 3.12.7):**
   - Ссылка: [https://www.python.org/downloads/windows/](https://www.python.org/downloads/windows/)
   - **Важно:** отметьте галочки:
     - *Use admin privileges when installing py.exe*
     - *add python.exe to PATH*
   - **Обязательно перезагрузите компьютер!**

3. **Настройка VSCode:**
   - Установите расширение **Jupyter**.
   - Откройте командную строку (Win + R → `cmd`) и установите необходимые библиотеки:
     ```bash
     pip install pandas numpy matplotlib scikit-learn jupyter
     ```

#### Запуск блокнота
1. Скачайте файл `.ipynb` (и опционально `.csv`) с Google Диска.
2. Откройте скачанный файл в VSCode.
3. Вверху нажмите **Run All**.
4. В появившемся меню выберите **Python Environments**.
5. Дождитесь, пока система подтянет установленный Python (или выберите его вручную).
6. Если появится запрос на установку `ipykernel` — соглашайтесь.
7. Дальнейшая работа аналогична Colab:
   - Запуск ячейки: кнопка слева от ячейки или `Ctrl + Enter`.
   - Запуск всего файла: кнопка **Run All**.
   - Новая ячейка: кнопка **+ Code**.

## Требования к зачёту

Для успешной сдачи зачета необходимо выполнить **3 шага**:

### 1. Пройти тест
- Перейдите по ссылке из файла `ТЕСТ.txt`.
- Сделайте **скриншот** результата прохождения теста.
- Сохраните скриншот в формате `.png` или `.jpeg`.

### 2. Выполнить зачетное задание
- Файл: `Зачётное_задание.ipynb`.
- Результат сохраните в формате `.ipynb` или `.py`.

### 3. Загрузить работу на Google Диск
- Зайдите в папку:  
  `Зачётное задание → Зачётные задания группы`
- **Создайте личную папку** с названием по образцу:

Фамилия_Имя_Отчество_Группа_зачетное_задание

- Внутри папки должно быть **минимум два файла**:
1. `Фамилия_Имя_Отчество_Группа_зачетное_задание.ipynb` — решение задачи.
2. `Фамилия_Имя_Отчество_Группа_ТЕСТ.png` — скриншот теста.

## Важные замечания
- Файлы `.csv` скачивать не обязательно, если вы работаете с оригинальными ссылками.

- Все изменения в Colab сохраняются автоматически при наличии копии на вашем Диске.
